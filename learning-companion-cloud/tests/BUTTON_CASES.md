# 孩子端四按钮验收用例

## 覆盖范围

本用例覆盖孩子端每张任务卡上的四个核心按钮：`开始`、`暂停`、`我做完了，开始检查`、`我卡住了`。每个按钮都按“目标 → 过程 → 结果”验收，自动化覆盖在 `scripts/self_test.py` 和 `scripts/ui_click_test.py`。

## BTN-START-001：开始

- 目标：孩子点击后明确进入当前任务，任务状态变为进行中，并开始累计学习时间。
- 过程：在孩子端选择一张未开始任务，点击 `开始`；等待 1 秒；刷新任务列表。
- 结果：当前任务状态显示 `进行中`；任务卡高亮；显示 `已学 0:01` 或继续增长的计时；接口返回 `status=in_progress`、`timer_state=running`、`last_started_at` 非空；其他任务不应被改为进行中。

## BTN-PAUSE-001：暂停

- 目标：孩子临时休息时停止计时，不丢失已累计学习时间。
- 过程：对正在进行中的任务点击 `暂停`；刷新任务列表。
- 结果：当前任务状态显示 `已暂停`；计时停止增长；接口返回 `status=paused`、`timer_state=stopped`、`elapsed_seconds` 保留已学秒数；再次点击 `开始` 可继续累计。

## BTN-COMPLETE-001：我做完了，开始检查

- 目标：孩子声明完成后进入检查阶段，并立即出现对应任务的小测/检查题。
- 过程：对任务点击 `我做完了，开始检查`。
- 结果：当前任务状态变为 `检查中`；计时停止；右侧 `小测 / 检查` 区域显示该任务标题和至少 3 道题；题目不暴露标准答案；提交后生成批改结果并同步家长端。

## BTN-STUCK-001：我卡住了

- 目标：孩子只把当前任务标记为卡住，并获得针对当前问题的分层提示。
- 过程：对某一张任务卡点击 `我卡住了`；输入具体问题，例如 `不认识鹭这个字`。
- 结果：只有当前任务状态变为 `卡住了`；其他任务状态不改变；计时停止；右侧显示 `当前卡住：任务标题` 和 `你的问题：不认识鹭这个字`；辅导内容应针对问题给出读音、含义、记法和下一步动作，例如 `鹭` 读 `lù`；家长端卡住任务数量增加并记录补漏项。

## 回归命令

```powershell
python .\scripts\self_test.py
python .\scripts\ui_click_test.py
```
