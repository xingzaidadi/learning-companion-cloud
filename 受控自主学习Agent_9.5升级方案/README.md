# 受控自主学习 Agent 9.5 升级方案

本文件夹用于承接 `C:\Users\MI\Desktop\AI学习任务\agent相关` 下全部 Agent 学习资料，并将这些资料和当前“11 岁孩子暑假自主学习陪跑系统”的 Agent 实现逐项对照，形成可落地的 9.5 分升级设计。

## 文档结构

1. `00_AGENT资料全集清单.md`：本地 Agent 资料全集清单，逐文件标注用途、当前匹配和后续动作。
2. `01_资料与当前Agent一一对照.md`：按 Agent 设计模式、Runtime、记忆、测试、CLI 等维度对照当前系统。
3. `02_后续使用方式_输入过程输出.md`：说明孩子、家长、Agent 后续如何使用，输入、过程、输出分别是什么。
4. `03_9.5分总体改造方案.md`：从产品、Agent、测试、UI、数据模型给出总体升级路线。
5. `04_受控自主学习Agent详细设计.md`：技术设计，包括 Agent 架构、工具注册、状态机、记忆系统、RAG、API、数据表。
6. `05_测试_UI_Agent多维验收设计.md`：从测试开发、UI 设计、Agent 开发三个角度制定验收标准。
7. `06_实施路线图与评分.md`：分阶段实施计划、验收门槛、预期评分。

## 一句话目标

把当前系统从：

```text
规则编排 + 局部 AI 调用 + 本地兜底
```

升级为：

```text
受控自主学习 Agent：家长设目标，孩子执行学习，Agent 自动规划、辅导、出题、批改、补漏、总结，但所有关键动作受规则和家长边界约束。
```
